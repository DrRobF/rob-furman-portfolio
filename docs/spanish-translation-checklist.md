# Spanish Translation Coverage Checklist

Use this developer-only checklist before and after edits that touch visible copy, H.E.L.P. pages, simulations, diagnostics, cards, buttons, or dashboard panels.

## Manual verification steps

1. Switch to Español.
2. Check homepage.
3. Check H.E.L.P. landing page.
4. Check diagnostic question screen.
5. Check 8 Factors course module.
6. Check Parent Call landing.
7. Check Urban Student Sim intro and at least one runtime scene.
8. Check Leadership Sim intro and at least one runtime scenario.
9. Check Dashboard headings/cards.
10. Switch back to English.

## Warning

Do not hardcode visible English strings in H.E.L.P. pages, simulation scene data, diagnostic questions, buttons, cards, or dashboard panels. Use the existing language helper/localized object pattern.
