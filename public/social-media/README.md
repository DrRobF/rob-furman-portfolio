# Social Media Graphics

This folder is for all social media graphics used in Content360.

Upload all quote graphics, project graphics, VIC graphics, simulation graphics, and reel/video thumbnails here.

- Do not create separate subfolders.
- Use clear file names to identify each graphic.

Example public URLs:
- https://www.drrobfurman.com/social-media/vic-teacher-support-01.png
- https://www.drrobfurman.com/social-media/leader-sim-pressure-01.jpg
- https://www.drrobfurman.com/social-media/student-sim-behavior-story-01.png

Note: In Next.js, files inside `/public` are served from the site root. For example, `public/social-media/example.png` is available at `/social-media/example.png`.
